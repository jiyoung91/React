map() 메서드는 배열 내의 모든 요소 각각에 대하여 주어진 함수를 호출한 결과를 모아 새로운 배열을 반환합니다.
   
        arr.map(callback(currentValue[, index[, array]])[, thisArg])
        arr.map(함수(현재값,인덱스){

        })
        arr.map(function(변수,변수){
            return 값
        })
        arr.map((변수,변수) => {
            /* {}사용하면 return 꼭 써야함 */
            return 값
        })
        /* 결과가 1개일때 :일급함수!!! */
        arr.map((변수,변수)=> 값 )

        매개변수
        callback :function
        currentValue : 현재값
        index Optional :인덱스
        
        반환 값
        배열의 각 요소에 대해 실행한 callback의 결과를 모은 
        새로운 배열!!!
